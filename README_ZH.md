> ⚠️ **重要提示**

## ✨ 核心特性

- **缓存系统**: 高性能数据缓存
- **文件存储**: 本地磁盘及云存储
- **文件模板**: Word 文档模板引擎

## 🆚 社区版 vs 商业版

| 功能模块 | 具体能力 | 🟢 社区版 | 🔵 商业版 |
| :--- | :--- | :---: | :---: |
| **缓存** | SQLite 缓存 | ✅ | ✅ |
| | Redis 缓存 | ❌ | ✅ |
| | TongRDS 缓存 | ❌ | ✅ |
| **存储** | 磁盘存储 | ✅ | ✅ |
| | 阿里云 OSS | ❌ | ✅ |
| | AWS S3 | ❌ | ✅ |
| | Cloudflare R2 | ❌ | ✅ |
| | MinIO | ❌ | ✅ |
| | 七牛云 | ❌ | ✅ |
| **文件模板** | Word 模板 | ✅ | ✅ |
| | Excel 模板 | ❌ | ✅ |
